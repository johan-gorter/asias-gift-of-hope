# Eastern Redesign Screenshots

Screenshots of the redesigned website with Eastern/Oriental styling and rounder elements.

## Files

- `01-homepage-hero.png` - Hero section with fully rounded button
- `02-homepage-full.png` - Full homepage showing all sections including new collection points
- `03-donation-page-full.png` - Donation page with rounded elements

## Key Visual Changes

- **Buttons**: Fully rounded (pill-shaped) with enhanced shadows
- **Cards**: Larger border radius (1.5rem) with elevation on hover
- **Images**: Very rounded corners (rounded-3xl)
- **Icons**: Circular containers instead of square
- **Gold borders**: Rounded with subtle glow effect
